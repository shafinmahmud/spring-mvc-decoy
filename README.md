Spring MVC Decoy
-------------------
Demonstrates the Spring MVC web framework through an imaginary business logic in a multi module way. You should have find all best practices that can be easily used as kicking template for a new project.

In this showcase you'll see the following in action:

* The simplest possible @Controller
* Mapping Requests
* Obtaining Request Data
* Generating Responses
* Message Converters
* Rendering Views
* Type Conversion
* Validation
* Forms
* File Upload
* Exception Handling

To get the code:
-------------------
Clone the repository:

    $ git clone git@github.com:shafinmahmud/spring-mvc-decoy.git

If this is your first time using Github, review http://help.github.com to learn the basics.

To run the application:
-------------------	
From the command line with Gradle:

    $ cd spring-mvc-decoy
    $ gradle tomcat:run 

Access the deployed web application at: http://localhost:8080/spring-mvc-decoy/
